# DevOps Track Submission

This is my submission for the DevOps track. I completed it as one project, so DeployReady is the main submission and the other challenge folders are part of the finished work. DeployReady includes the Terraform bonus work, which ties together most of the other challenge pieces.

- **DeployReady**: Containerisation, CI/CD pipeline, and cloud deployment. The bonus project I did here was Terraform for the deployment infrastructure. See [DeployReady/README.md](DeployReady/README.md) and [DeployReady/DEPLOYMENT.md](DeployReady/DEPLOYMENT.md).
- **InfraBlueprint**: Terraform infrastructure-as-code for a two-tier application. The bonus project I did here was modules, and the README covers the setup and decisions I made. See [InfraBlueprint/README.md](InfraBlueprint/README.md) and [InfraBlueprint/infra/modules](InfraBlueprint/infra/modules).
- **WatchTower**: Observability stack (Prometheus, Grafana) integrated with three services. The bonus project I did here was the uptime graph in Grafana. See [WatchTower/README.md](WatchTower/README.md) and [WatchTower/app](WatchTower/app).
